Список задач для доработки
Проверить что JWT токен Directus обновляется автоматически перед истечением текущего
Убрать вкладку "Прошедшие"
Удалить вкладку "Прошедшие" из раздела "Запланированные".
Массовое добавление источников
Добавить возможность загружать источники списком (15-20 шт.) без ручного ввода.
Названия каналов можно брать из исходных данных автоматически.
Исправление добавления изображений
При добавлении второго и последующих изображений кнопка "Сгенерировать" должна появляться у нового поля, а не оставаться у первого.
Сейчас пользователи сталкиваются с неочевидным поведением: при генерации нового изображения ссылка в первом поле перезаписывается.
Корректировка аналитики
Исправить некорректные данные в разделе "Аналитика":
Количество опубликованных за 7 дней постов.
Завышенные показатели просмотров.
Решение проблемы с VPN для Gemini 2.5
Обеспечить стабильную работу Gemini 2.5 без необходимости использования VPN.
Скрыть раздел "Настройки"
Скрыть раздел "Настройки" для всех пользователей.
Новые аккаунты должны использовать ключи ИИ-сервисов по умолчанию.
Перевод промтов для генерации изображений
Добавить кнопку перевода русского промта на английский перед генерацией изображения.
Или автоматически переводить промт при отправке запроса.
Решить проблему генерации изображений с русским текстом (например, "Дайджест" на зеленом фоне).
Выкладка сторис
Добавить функционал публикации сторис в соцсетях.
Постинг в YouTube
Реализовать возможность публикации видео и подписей к ним на YouTube.
Очистка TG-тегов для других платформ (n8n)
TG-теги (<blockquote>, <code>, <pre>) должны автоматически удаляться для ВК, Instagram, FB, но оставаться для Telegram.
Исправление сбора трендов
Решить проблему, когда "Сбор трендов" запускается, но не возвращает результаты.
Возможно, уже доработано (уточнить у Романа).
Упрощение ввода API-ключей
Предложить альтернативный способ авторизации (например, через логин/пароль от соцсети) вместо ручного ввода ключей API.
