Создай веб-приложение на React + Node.js/Express со следующим функционалом:

### 1. **Аутентификация через Directus**
- Страница входа с email/password через Directus SDK
- Сохранение access token в localStorage
- Защищенные роуты для авторизованных пользователей
- Навигация: Keywords, Campaigns, Analytics, Logout

### 2. **Управление кампаниями**
- Таблица кампаний (название, описание, дата создания)
- Кнопка "+ Add Campaign" (модальное окно с формой)
- Привязка ключевых слов к кампаниям через Directus Collections
- Опции: View Keywords, Delete, Export (CSV/PDF)

### 3. **Интеграция с WordStat**
- Поиск ключевых слов через API xmlriver.com/wordstat/json
- Отображение: ключевое слово, тренд, конкуренция
- Кнопка "Add to Campaign" с выбором целевой кампании
- Валидация ключевых слов (макс. длина, стоп-слова)

### 4. **Аналитика**
- Дашборд с графиками (ApexCharts):
  - Динамика трендов (линейный график)
  - Распределение аудитории: гео/возраст/пол (столбчатая диаграмма)
  - Топ ключевых слов по CTR (круговая диаграмма)
- Фильтры: период, кампания, регион

### 5. **Генерация контента через DeepSeek**
- Кнопка "Generate Content" в карточке кампании
- Запрос к DeepSeek API с параметрами:
  ```json
  {
    "keywords": ["..."],
    "tone": "профессиональный",
    "length": 500,
    "instructions": "SEO-текст с LSI-словами"
  }
Модальное окно для предпросмотра/редактирования текста

Автосохранение в Directus через linked collection
